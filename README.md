# OnlyMsg
### Supports major all major Socials and other messaging platforms
### Why? Made it because I wanted to message on socials without falling to the "Feed" and aggregate other messaging platforms that can be useful for a university student including Microsoft Teams and Google Classroom.

### I hope this helps out everyone who want to quit social media but still wanna message people.
Currently only available for MacOS and Windows. I'm working on the Android versions so that you can finally delete your social media apps. Sadly, for iOS there won't be an app because of strict rules from side of Apple.
